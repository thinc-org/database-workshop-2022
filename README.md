## Get start !

clone project template in to your workspace
```bash
git clone https://github.com/thinc-org/database-workshop-2022.git
```

if you haven't installed `pnpm` yet.
```bash
npm install -g pnpm
```

then run
```bash
cd database-workshop-2022

pnpm install
```
